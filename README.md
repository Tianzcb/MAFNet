# MAFNet


------

计算机辅助诊断系统已应用到肝脏局灶性病变（Focal Liver Lesions，FFLs）的检测和分类，有助于放射科医生临床诊断。
### 一. **所需环境**
> * Anaconda3（建议使用）
> * python3.6/3.7/3.8
> * pycharm (IDE)
> * pytorch 1.10 (pip package)
> * torchvision 0.11.1 (pip package)



### 二. **训练流程** 
### 1. 数据预处理
> * dataset.py
> * transformer.py


### 2. 训练

> * MAFNet.py
